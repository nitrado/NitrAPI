If you are a Twitch Streamer you can use our Twitch API Endpoint to check if a player on your server is a follower or subscriber. This feature can be used for subscriber or follower only servers.

/twitch/allowed_connect

Checks if a specified player follows our subscribed your Twitch channel.

Request:

GET /twitch/allowed_connect?access_token=<access_token>

GET Parameters:

account_type=<steam|battlelog_pc|minecraft>
account_id=<steam_id|battlelog_id|minecraft_uuid>

Response:

{
   "status":"success",
   "data":{
      "streamer":"gronkh",
      "user":{
         "twitch_name":"moobot",
         "type":"minecraft",
         "id":"myuuid"
       },
       "subscribed":true,
       "follows":true
      }
   }
}
